MEUS PROJETOS.

BoxBalancer

O BoxBalancer é um sistema de controle de caixa diário projetado para gerenciar movimentações de dinheiro em todas as formas de pagamento. Desenvolvido com C#, .NET e SQL Server, o sistema facilita o controle das entradas e saídas de valores, enquanto a interface visual, criada com Windows Forms, oferece uma experiência prática e intuitiva para a gestão financeira diária.

ExamList

Desenvolvido com HTML, CSS e MySQL, este sistema permite o registro e a visualização de resultados de exames médicos. Após inserir os dados, o sistema gera um relatório completo com todos os resultados, oferecendo uma visão clara e organizada das informações.

Bot Whatsapp

Desenvolvi um bot usando Python, Selenium e WebBrowser para automatizar cobranças de IPTV. O bot envia notificações e lembretes de pagamento de forma eficiente, reduzindo o tempo gasto em tarefas manuais e melhorando a comunicação com os clientes.
